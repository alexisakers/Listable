EmbeddedList

public struct EmbeddedList : ItemContent

Describes item content which can be used to embed a list inside another list, for example if you’d like to place a horizontally scrollable list within a vertically scrolling list, or vice versa.

You rarely use this type directly. Instead, use the static .list function on Item.

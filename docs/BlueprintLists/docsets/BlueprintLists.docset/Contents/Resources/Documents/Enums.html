  • ListSizing

    Provides the possible options for how to size and measure a list when its measured size is queried by the layout system.

    You have two options: .fillParent and .measureContent.

    When using .fillParent, the full available fitting size will be taken up, regardless of the size of the content itself.

    When using .measureContent, the content will be measured within the provided fitting size and the smallest of the two sizes will be returned.
